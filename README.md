# NIOKR Analysis

Проект посвящен анализу и визуализации данных о научно-исследовательских и опытно-конструкторских работах (НИОКР).

В рамках проекта выполняется автоматическая обработка данных о научных проектах, анализ организаций-исполнителей, направлений исследований, типов работ и динамики научной активности. На основе полученных данных строятся аналитические графики и формируется итоговый отчет.

---

## Функционал проекта

### Предобработка данных

* Загрузка данных из CSV-файла;
* Очистка и проверка структуры датасета;
* Обработка числовых показателей финансирования;
* Удаление пропусков и подготовка данных к анализу.

### Аналитическая обработка

* Подсчет количества НИОКР по организациям;
* Анализ распределения финансирования по научным направлениям;
* Исследование структуры проектов по типам работ;
* Анализ динамики научных проектов по годам.

### Визуализация

Автоматическая генерация и сохранение графиков в формате PNG:

* Топ организаций по количеству проектов;
* Топ направлений по объему финансирования;
* Структура типов работ;
* Динамика количества НИОКР по годам.

### Отчетность

Автоматическое формирование аналитического отчета в формате Markdown.

## Визуализации

### 1. Топ-10 организаций по количеству НИОКР

Отображает организации с наибольшим количеством зарегистрированных научно-исследовательских проектов.

### 2. Топ направлений по финансированию

Показывает направления исследований с наибольшим суммарным объемом финансирования.

### 3. Структура проектов по типам работ

Круговая диаграмма распределения проектов между НИР, ОКР и технологическими работами.

### 4. Динамика НИОКР по годам

Линейный график изменения количества зарегистрированных проектов по годам.

## Как запустить проект

### Установка зависимостей

```bash
pip install pandas matplotlib seaborn
```

или

```bash
pip install -r requirements.txt
```

### Запуск анализа

Убедитесь, что файл `niokr.csv` находится в одной директории со скриптом `main.py`.

Запустите:

```bash
python main.py
```

После завершения работы будут созданы:

* папка `niokr_plots` с графиками;
* файл `niokr_report.md` с аналитическим отчетом.

## Результаты анализа

Проект позволяет получить наглядное представление о структуре научно-исследовательской деятельности, распределении финансирования между направлениями исследований и активности организаций-исполнителей НИОКР.

Полученные результаты могут использоваться для учебных, исследовательских и демонстрационных целей в области анализа данных и визуализации информации.
